# 코골이 앱

건강한 수면을 위한 코골이 모니터링 및 관리 애플리케이션

## 기술 스택

- React 18
- TypeScript
- CSS3

## 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm start
```

브라우저에서 [http://localhost:3000](http://localhost:3000)를 열어 앱을 확인할 수 있습니다.

### 빌드

```bash
npm run build
```

### 테스트

```bash
npm test
```
